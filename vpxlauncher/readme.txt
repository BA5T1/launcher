Installation:

start VPX-LauncherInstaller-x.xx.exe to install into
destination folder.

there is a new green vpx-ixon at your taskbar.
rightclick to see menu

setup your tables-folder
set "Autostart last table" (for next start)
click on apply changes

click on "Launch VPX"

launcher will pup up

----------------------------------------------------------

How to use:

key-mapping of each function is displayed 
at the bottom of launcher overlay
